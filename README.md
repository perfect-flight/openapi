# Perfect Flight — API Reference

Documentação pública da API da Perfect Flight, publicada em
**[docs.perfectflight.com.br](https://docs.perfectflight.com.br)** via GitHub Pages.

A fonte é o arquivo [`openapi.yaml`](openapi.yaml) (OpenAPI 3.0.4). O
[`index.html`](index.html) é a página estática gerada a partir dele com
[Redoc](https://github.com/Redocly/redoc) — **não edite o HTML à mão**.

## API

- **Auth:** OAuth2 client credentials (M2M).
- **Servidores:** sandbox (`https://api.sandbox.perfectflight.com.br/v1`) e
  produção (`https://api.perfectflight.com.br/v1`).
- **Contato:** developer@perfectflightapp.com
